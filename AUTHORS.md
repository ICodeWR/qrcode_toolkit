## 特别感谢

感谢以下开源项目，它们的优秀代码和理念帮助我们构建了 QR Toolkit：

- [PySide6](https://doc.qt.io/qtforpython/) - Qt for Python，提供强大的 GUI 框架
- [qrcode](https://github.com/lincolnloop/python-qrcode) - 纯 Python 二维码生成库
- [segno](https://github.com/heuer/segno) - 支持 SVG 输出的二维码生成库
- [pyzbar](https://github.com/NaturalHistoryMuseum/pyzbar/) - 二维码解码库
- [OpenCV](https://opencv.org/) - 计算机视觉库，用于摄像头图像处理
- [Pillow](https://python-pillow.org/) - Python 图像处理库
- [NumPy](https://numpy.org/) - 数值计算库
- [pytest](https://pytest.org/) - 测试框架
- [uv](https://github.com/astral-sh/uv) - 极速 Python 包管理工具

## 如何成为贡献者

我们欢迎任何形式的贡献！您可以通过以下方式参与：

1. **提交代码** - 修复 bug 或添加新功能
2. **改进文档** - 完善使用说明、翻译文档
3. **报告问题** - 发现 bug 或提出功能建议
4. **分享项目** - 向他人推荐 QR Toolkit

请阅读 [CONTRIBUTING.md](CONTRIBUTING.md) 了解详细信息。
